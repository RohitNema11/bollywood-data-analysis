# Bollywood Movies Data Analysis and Visualization

## Project Overview
The project explores a dataset of Bollywood movies to analyze patterns and trends in the film industry. Using exploratory data analysis techniques and visualizations, the project highlights insights related to movie ratings, genres, release patterns, etc.

## Analysis Performed
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Genre distribution analysis
- Movie rating analysis
- Visualization of trends using charts and graphs

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
